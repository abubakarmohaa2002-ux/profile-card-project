# Profile Card Project

A simple profile card webpage built with HTML and CSS to demonstrate the use of basic CSS selectors and Flexbox layout techniques.

## Features

- Universal Selector (`*`)
- Element Selectors (`body`, `h1`)
- Class Selectors (`.card`, `.role`, `.bio`)
- ID Selector (`#name`)
- Centered Card Layout using Flexbox
- External CSS Styling
- Clean and Responsive Design

## Technologies Used

- HTML5
- CSS3

## Project Preview

The webpage displays:
- User Name
- Job Role
- Short Biography
- Follow Button

All content is styled using CSS selectors and positioned in the center of the page.
